◆ BinaryToCSV()

std::string TKFILEUTIL::BinaryToCSV ( std::string  file_path,
bool  force_over_write = false 
)

YOKOGAWA DL シリーズの BINARY ファイルを ASCII CSV ファイルへ変換します。

引数
[in]file_path対象となるBINARYファイルパスです。拡張子を含みます。
[in]force_over_write変換後のファイルと同じ名前のファイルが既に存在した場合、変換を行うかを指定します。 デフォルトではfalseで、変換は行われません。
戻り値
生成されたCSVファイルパスが返されます。拡張子を含みません。
覚え書き
変換が実行されたかによらず変換後のファイル名が返されます。

◆ ExtractWDF()

std::string TKFILEUTIL::ExtractWDF ( std::string  file_path,
bool  force_over_write = false 
)

YOKOGAWA DL850 の WDF ファイルを WVF + HDR ファイルへ変換します。

引数
[in]file_path対象となるWDFファイルパスです。拡張子を含みます。
[in]force_over_write変換後のファイルと同じ名前のファイルが既に存在した場合、変換を行うかを指定します。 デフォルトではfalseで、変換は行われません。
戻り値
生成されたWVFファイルパスが返されます。拡張子を含みません。
覚え書き
同時に出力されるHDRファイルは波形情報を取得する際に必要となります。
変換が実行されたかによらず変換後のファイル名が返されます。
