ADCモデル列挙型

ADCモデル名はADCの型番を指します(多くの場合は商品名と呼べるかもしれません)。
ADCモデルはあるADCについての固有な名前を意味しません。従って複数の同じADCがシステムに組み込まれている場合は、 それら複数のADCが同じADCモデル名を持ちます。
ADCモデル名という呼称は横河電機の流儀に従ったものです。
