◆ Delete()

int TKADCCONTROL_DL850::Delete ( std::string  file_name)
virtual

ADCの内部ストレージから指定したファイル名のバイナリデータを削除します。

存在しないファイル名が指定された場合の挙動はADCの仕様によります。

引数
[in]file_name削除ファイル名を指定します。拡張子は不要です。
戻り値
0: 正常終了
0以外: tmctl.hで定義されたエラーコードを返します
覚え書き
DL750, DL850 では存在しないファイル名が指定された場合、ADCの画面に警告が出るに留まり、コントロールは継続されます。
削除が実行されるとディスクアクセスが発生するためWaitADC()等で待つ必要があります。
